## sublime
### 常用功能
+ File=> new File  创建文件
+ File=> open File  打开文件
+ File=> open Folder  打开文件夹

### 安装插件
> emmet 可以为html代码提供代码补全提示.
+ 打开控制台 ctrl+shift+p 
+ 查看安装列表  list packages
+ 如果没有emmet插件,需要安装 ctrl+shift+p ==> install packages 
+ emmet 回车,安装完毕会有提示.
